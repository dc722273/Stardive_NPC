# Godot engine

本项目固定使用 Godot `4.6.3-stable` standard Windows x86_64 版本。

常用入口：

```powershell
.\tools\godot\open-editor.cmd
.\tools\godot\godot.cmd --version
.\tools\godot\run-tests.cmd
```

本机 engine 二进制位于 `tools/godot/Godot_v4.6.3-stable_win64*.exe`，下载归档和 checksum 位于 `tools/godot/downloads/`。这些本机依赖已在 `.gitignore` 中排除。
